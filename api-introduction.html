Welcome to the API Introduction

This API provides endpoints for managing characters in a database. You can perform various operations such as creating, updating, deleting, and retrieving characters.

Endpoints:

  • GET /chars: Get all characters
  • GET /chars/{name}: Get a character by name
  • POST /chars: Create a new character
  • PATCH /chars/{name}: Update a character by name
  • DELETE /chars/{name}: Delete a character by name

For more details on how to use each endpoint, please refer to the API documentation.
